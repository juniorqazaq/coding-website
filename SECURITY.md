# Security Policy

## Supported Versions

The following versions of Tamasha Learning Platform are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Tamasha Learning Platform seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please report them by sending an email to [security@tamasha-learning.com](mailto:security@tamasha-learning.com).

### What to Include
To help us prioritize and research the issue, please include as much information as possible:
- A detailed description of the vulnerability.
- Steps to reproduce the issue (PoC scripts or screenshots are helpful).
- Potential impact of the vulnerability.
- Any suggested mitigations.

## Response Policy

- **Acknowledgment:** You will receive an acknowledgment of your report within 48 hours.
- **Verification:** We will work to verify the vulnerability and determine its severity.
- **Resolution:** If confirmed, we will develop a fix and coordinate a release.
- **Disclosure:** We follow a coordinated disclosure policy. We ask that you do not disclose the vulnerability publicly until a fix has been released.

We appreciate your help in keeping Tamasha Learning Platform secure for everyone.
